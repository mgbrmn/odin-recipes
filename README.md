# Odin Recipes

This project is part of The Odin Project Foundations course. It is a basic HTML website that displays a collection of recipes.

For this project, I followed the instructions as provided in The Odin Project without adding additional features or modifications. Therefore, my focus was to apply the fundamental concepts I am currently learning, particularly in basic HTML structure and workflow.

As I continue progressing through the course, I plan to revisit and improve this project by applying new skills and concepts I learn along the way.

- Using Git for version control (basic add, commit, and push workflow)
