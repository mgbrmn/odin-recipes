# Odin Recipes

## Project Description
This project is part of The Odin Project Foundations course. It is a basic HTML website that displays a collection of recipes.

## Skills Demonstrated
By completing this project, I practiced:

- Writing basic HTML structure
- Using headings, paragraphs, and lists
- Creating links between pages
- Organizing files and folders
- Using Git for version control (basic add, commit, and push workflow)
